## Adding a new post

On github.com, open the `docs/posts` folder, click **Add file → Create new
file**, and name it something like `2026-08-01-my-title.md`. Start the file
with a header block like the one this post uses:

```
---
date: 2026-08-01
categories:
  - Blackjack
---

# The actual title readers see

Opening paragraph, which becomes the excerpt...

<!-- more -->

The rest of the post.
```

Commit the file and the site rebuilds itself in about a minute. Categories are
created automatically the first time you use them — `Essays`, `Blackjack`, and
`Lapilli` will each get their own index page once a post uses them.

## Formatting

Posts are plain Markdown. Footnotes work,[^1] as do blockquotes:

> A quotation set off from the main text.

Headings inside a post (like the ones above) show up in the post's own
structure but keep the front page clean.
